# Angry Birds

Angry Birds is a fun game that skips the pigs and collects golden eggs.

## Features
* Controlling Bird Smoothly
* Supports Touch Screens
* Highest Score Tracking 
* Pure Code
* Animations
* Fast Loading
* Cross-Browser
* Fully Responsive
* Much More
